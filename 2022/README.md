### Prerequisites

- [Clojure](https://clojure.org/releases/downloads)
- [Leiningen](https://leiningen.org/)


### Usage

To run code

```bash
$ lein run
```

To run unit tests

```bash
$ lein test
```

### Options

`lein run {n-day} ..` to run only the chosen days

### Examples

```bash
$ lein run 1 2
$ ...
$ ---- day 1 ------
$ part 1:  66306
$ part 2:  195292
$ ---- day 2 ------
$ part 1:  12794
$ part 2:  14979
```
